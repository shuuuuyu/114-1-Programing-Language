## 期末專題連結
TripStarter 🚀

**給嚮往遠方的你，來一趟說走就走的旅行**

TripStarter 是一個針對自由行旅遊族群設計的 AI 行程規劃工具，
讓你 **不用花時間查資料**，只要輸入地區、預算、天數與旅遊偏好，
就能快速生成完整自由行行程建議。

[點此查看專題repository](https://github.com/41371122h-lichi/programming_group)

or 複製連結 https://github.com/41371122h-lichi/programming_group
## 功能特色

* **測試旅遊類型**：簡單填一下問卷，看看你適合什麼樣的旅遊方式。
* **一鍵生成行程**：輸入預算、天數與國家，讓AI幫您安排地點及行程！可 PDF 輸出，方便儲存或分享行程。
* **AI 智慧修正**：利用Gradio的界面匯入已經安排好的行程，再串接Gemini API對行程進行優化，如：交通時間與動線、天氣應對與備選方案，讓行程更加的流暢。
* **探索周邊景點**：連結地圖相關的API，如：Serp, Folium等，然後做下拉式選單讓使用者選擇，如：餐廳、咖啡廳等，去看在旅遊中有哪些地方在景點附近，給使用者做快速決策。
